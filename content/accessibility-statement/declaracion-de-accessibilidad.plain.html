Declaración de accesibilidad para los sitios web de AbbVie

Esta es una declaración de accesibilidad de AbbVie Inc.

English version

Tenemos el compromiso de asegurar que los sitios web de AbbVie sean accesibles a todos los usuarios, incluidos a los que usan tecnología asistencial, como lectores de pantalla. Nuestros sitios web se actualizan con regularidad y nos esforzamos continuamente por asegurar la accesibilidad y mejorar la experiencia del usuario. Si tiene dificultad al accesar alguna sección de nuestros sitios web o si desea dejarnos saber su sentir sobre la funcionalidad de nuestros sitios web, comuníquese con nosotros por medio de uno de los métodos de contacto que aparecen a continuación. Tomaremos en consideración su sentir al evaluar las maneras de mejorar la experiencia del usuario y hacer ajustes, y comunicarnos de manera efectiva con todos los visitantes del sitio web.

Contacto para comunicarse con nosotros

Agradecemos la información que nos provea respecto a la accesibilidad de AbbVie.com y otros sitios web de AbbVie. Déjenos saber si enfrenta alguna situación de accesibilidad de algún sitio web de AbbVie o si tiene una sugerencia en cuanto a cómo mejorar la accesibilidad o uso de un sitio web comunicándose con nosotros por medio de uno de los siguientes métodos. Haga referencia al sitio web particular y describa la característica específica a la que entiende que no puede tener acceso completo o sugiéranos cómo podemos mejorar:

  • Formulario de contacto
  • Correo electrónico: webaccessibility@abbvie.com
  • Dirección postal: 1 North Waukegan Road, North Chicago, IL, 60064

Medidas para apoyar la accesibilidad

AbbVie Inc. toma las siguientes medidas para asegurar la accesibilidad a nuestros sitios web:

  • Incluye accesibilidad en nuestras políticas internas.
  • Asigna un oficial y/o mediador de accesibilidad

Especificaciones técnicas

La accesibilidad de nuestros sitios web depende de las siguientes tecnologías para trabajar con la combinación particular de navegadores web y cualquier tecnología asistencial o complementos (plugins) instalados en su computadora:

  • HTML
  • WAI-ARIA
  • CSS
  • JavaScript

Estas tecnologías se utilizan para cumplir con las normas de accesibilidad usadas.

Functional Requirements & Acceptance Criteria

Hero Banner (4 variations)

Video Hero

Autoplays muted background video with heading overlay. Includes pause/play toggle. Falls back to static image on mobile.

  • Video autoplays muted on desktop (min-width: 1024px)
  • Static image displayed on mobile (max-width: 743px)
  • Pause/play button toggles video playback
  • H1 heading overlays with readable contrast
  • Full viewport width on all breakpoints

Image Hero

Full-width hero image with centered text overlay and optional CTA button.

  • Image spans full viewport width
  • Text overlay centered with contrast ratio ≥ 4.5:1
  • Responsive image via Scene7 $Hero$ preset
  • H1 in F37 Lineca Medium 40px/44px white
Card Grid (3 variations)

Three Column Cards

  • 3 cards in equal columns at desktop (min-width: 1024px)
  • Stacks vertically on mobile (max-width: 743px)
  • Entire card area clickable
  • Hover shows visual feedback (shadow/color)
  • Images maintain consistent aspect ratio

Article Cards

  • 3-column grid at desktop
  • Category tag as colored label
  • Date as "Month DD, YYYY"
  • Read time as "X Minute Read"
  • Cards fully clickable to article URL
Accordion (3 variations)

FAQ Accordion

  • All items collapsed by default
  • Click header toggles expand/collapse with smooth animation
  • "Expand All" opens all items simultaneously
  • aria-expanded attribute toggles correctly
  • Keyboard accessible (Enter/Space)

Medicine List Accordion (50+ items)

  • Global Expand All / Collapse All for 50+ items
  • EN and ES PDF download links per item
  • PDF links open in new tab
  • Performance smooth with 50+ items
Two Column Feature (3 variations)
  • 50/50 layout at desktop, stacks on mobile
  • Image Left: image left, text+CTA right
  • Image Right: text+CTA left, image right
  • Video Left: video thumbnail+play on left, text on right
  • CTA renders in uppercase blue style
Video Embed (2 variations)
  • Poster image with play button overlay
  • Duration badge visible (e.g. "Watch 3:49")
  • Click triggers video playback
  • Player controls visible during playback
  • Full-width on mobile
Exit Warning Modal
  • Triggers on click of external links (not whitelisted)
  • "No, I disagree" closes modal, stays on page
  • "Yes, I agree" opens external URL (new tab)
  • Overlay backdrop
  • Focus trapped within modal
  • Escape key closes modal
  • role=dialog, aria-modal=true
Profile Card Grid & Leader Bio

Profile Grid

  • 2-3 column responsive grid
  • Square headshot per card
  • "Meet [Name]" link to bio page
  • Single column on mobile

Leader Bio

  • Square headshot prominently displayed
  • Name as H1, title as subtitle
  • Multi-paragraph biography in body text
  • LinkedIn link if provided
  • Narrow content width, single column
Breadcrumb & Article Metadata

Breadcrumb

  • Full path from section root to current page
  • Separator between levels
  • All ancestors are clickable links
  • Current page as plain text
  • nav element with aria-label="breadcrumb"

Article Metadata

  • Date as "Month DD, YYYY"
  • Category as clickable tag
  • Reading time as "X Minute Read"
  • Horizontal layout with separators
